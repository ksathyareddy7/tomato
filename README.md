## Setup Steps
1. Clone the repo and open terminal in the project folder
2. run `npm install`
3. run `npm start`
4. scan the render QR in terminal using `Expo Go` app
5. MAKE SURE YOU ARE ON THE SAME WIFI NETWORK AS YOUR SYSTEM.
